Github

Création d'un nouveau repository

  1. git init (on initialise le repository)
  2. git add .* (ajout toutes les modification)
  3. git commit -m "first commit" (un fait le commit)
  4. git branch -M main (la branch master est main)
  5. git remote add origin https://github.com/jc-mairot-formation/STUD_Github.git (on lie le repository au projet)
  6. git push -u origin main (on pousse le projet sur le repository de github)

Routine pour push les données sur un repository

  1. git remote add origin https://github.com/jc-mairot-formation/STUD_Github.git
  2. git branch -M main
  3. git push -u origin main
